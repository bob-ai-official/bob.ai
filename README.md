bob.ai

Bob AI is a simple non-AI chatbot.
It is created to make learning Python as simplest as it can.
This project is published on MIT licence.

In the newest version of Bobs there is information who made it, when you're keeping it with no changes, please keep the information about author stock too.

We highly reccomend using Visual Studio Code for executing our program.
Online python executors mostly cannot all of our plugins.

If  using version 2.5 and higher make sure files are in this order:

1. bob_core.py
2. bob_gui.py (additional)
3. Bob2.5.py (And higher versions)

Have fun!
